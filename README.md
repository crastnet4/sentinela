========================================
              SENTINELA V1
========================================

Ferramenta desenvolvida em Python com foco educacional
para estudo de redes, requisições HTTP assíncronas
e automação via linha de comando.

----------------------------------------
FUNCIONALIDADES
----------------------------------------
[>] Interface CLI interativa
[>] Requisições HTTP utilizando aiohttp
[>] Execução de múltiplas tarefas assíncronas com asyncio
[>] Integração com webhooks
[>] Simulação de envio de requisições para testes em ambientes controlados

----------------------------------------
TECNOLOGIAS UTILIZADAS
----------------------------------------
- Python 3
- asyncio
- aiohttp
- requests
- colorama

----------------------------------------
INSTALAÇÃO
----------------------------------------
git clone https://github.com/ogredzin/sentinela.git
cd sentinela
pip install -r requirements.txt

----------------------------------------
USO
----------------------------------------
python main.py
